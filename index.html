Funcion validar rut

En esta seccion se ejecuta la llamada a la API para la validacion de un RUT. Ingrese su rut sin puntos ni guion en el input RUT, seguido por el digito verificador en el input DV.

Funcion Split

En esta seccion se ejecuta la llamada a la API para la separacion de nombres y apellidos. Ingrese su nombre completo en el campo Nombre y presione el boton.
